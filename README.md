# MemeMaster

A premium Meme Generator and WhatsApp Sticker creator.

## Cara Menggunakan

1. Buka file `index.html` di browser Anda (Chrome, Firefox, Edge, dll).
2. Upload gambar dengan cara drag & drop atau klik area gambar.
3. Tulis teks meme (Teks Atas / Teks Bawah).
4. Atur ukuran dan warna teks.
5. Pilih "Mode Stiker WA" untuk otomatis resize ke format stiker (512x512).
6. Download hasilnya!

## Fitur
- **WhatsApp Sticker Ready**: Output 512x512 .webp
- **Premium UI**: Desain modern dengan glassmorphism.
- **Privacy Focus**: Semua proses terjadi di browser Anda (offline-first), tidak ada gambar yang diupload ke server.
